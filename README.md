# Gradient Background Generator

A simple tool for generating linear-gradient backgrounds with live previews and editable color inputs.

## Features

- Real-time gradient preview
- Two color inputs with instant updates
- Random color generator button
- Displays active CSS gradient
- Minified CSS & JS for faster performance

## Files

- `index.html` – main UI and layout
- `style.css` / `style.min.css` – styling
- `script.js` / `script.min.js` – gradient logic and events
- `favicon.png` – project icon

## Usage

1. Select two colors or use **Randomize Colors**.
2. Copy the displayed CSS gradient value.
3. Use the gradient in your own projects.
